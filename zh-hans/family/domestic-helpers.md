# 香港外佣（家庭佣工）指南：招聘、合同、薪资规定与常见陷阱

香港是全球外籍家庭佣工制度最成熟的地区之一。目前在港外佣超过34万人，主要来自菲律宾和印度尼西亚，受《雇佣条例》及专项外佣政策保障。对于内地来港专才家庭——尤其是双职工家庭或有幼儿/长者照料需求的——聘用外佣是最直接有效的家庭支援方案之一。

## 为什么内地专才家庭普遍雇用外佣

内地专才来港后，通常面临以下现实：原有的家庭支援网络（父母、保姆资源）留在内地，香港本地托儿服务供应紧张，而工作时间往往比内地更长。外佣制度恰好填补了这一缺口。

| 需求场景 | 外佣能解决的问题 |
|---|---|
| 双职工有幼儿 | 全职照顾子女，接送上学，准备餐点 |
| 有长者同住 | 日常陪护、服药提醒、医院陪诊 |
| 繁忙专业人士 | 家务管理、采购、烹饪 |
| 有特殊需要子女 | 专项照料和活动支援 |

**费用效益**：2026年雇用一名外佣的月均总成本约港幣7,000–8,500元（含薪资、食物津贴、强积金供款），远低于聘用本地兼职帮工或私人托儿中心的费用。

## 2026年现行法定薪资与福利标准

香港法律对外佣待遇有明确规定，以下为2026年现行标准：

| 项目 | 法定标准（2026年） |
|---|---|
| 最低允许工资（MAW） | 港幣 **4,990元/月** |
| 食物津贴（如不提供膳食） | 港幣 **1,236元/月** |
| 每周休息日 | 每周至少1日，通常为周日 |
| 法定假日 | 每年12个公众假期（须放假或补假） |
| 年假 | 首年7日，随年资递增至最高14日 |
| 每年回乡机票 | 雇主须负担1张经济舱机票（合约届满或续约时） |
| 强积金供款 | 雇主须供款雇员薪酬5%（最低港幣 250元/月） |

**实际薪资**：虽然法定最低工资为港幣4,990元，但受供需影响，具备特定经验（幼儿照料、老人护理、烹饪技能）或香港工作经验的外佣实际月薪常在港幣5,500–7,000元之间。

## 招聘途径比较

| 途径 | 成本 | 优势 | 劣势 |
|---|---|---|---|
| **持牌外佣中介** | 港幣3,000–8,000（一次性） | 协助审查证件、合同准备、语言协调 | 质量参差，部分收费不透明 |
| **直接从海外招聘** | 较低（仅签证/机票等） | 可自行把关挑选 | 流程繁琐，雇主须自行处理入境文件 |
| **转雇在港外佣** | 较低 | 可当面访谈，了解既往雇主评价 | 须确认其现有合约已正式终止，两年规定不适用 |
| **网上平台（如HelperChoice、Enlist.com.hk）** | 低 | 履历浏览方便，可筛选技能/语言 | 需自行核实背景，无中介担保 |

**持牌中介**是初次聘用的新手家庭最常选择的方式，但务必核实中介是否持有劳工处颁发的牌照（可在劳工处网站查核）。

## 标准外佣合同主要条款

香港政府规定外佣合同须使用标准格式（ID 407），主要内容包括：

- **合约期限**：通常2年，期满双方协议可续签
- **工作地点**：合同须列明雇主住所地址，外佣须在该地址居住
- **工作职责**：家务及所列家庭成员的照料，不可派往其他住所工作
- **终止条款**：任一方须提前1个月书面通知，或支付1个月薪酬代替通知期
- **医疗保障**：雇主须为外佣购买医疗保险（涵盖住院及门诊）

合同须经香港入境处审批，并向劳工处备案方为有效。

## 住宿安排：雇主的法律责任

香港法律要求**雇主须为外佣提供免费住宿**，且住宿须符合合理居住标准。这是许多内地专才家庭初来乍到时未必意识到的法律义务。

**实际安排**：
- 大多数家庭为外佣提供独立工人房（若单位设有）
- 若单位无独立工人房，可安排有私隐保障的专属睡眠空间（如屏风隔断的区域）
- **不可**以金钱补偿替代住宿提供
- 与外佣商定不住宿安排并签署豁免声明在法律上无效，雇主仍须承担责任

若单位确实无法提供适当住宿，可向入境处申请豁免，但批准率较低，需提供合理理由。

## 外佣续签与转雇流程

**续签**：合约到期前2–3个月向入境处提交续签申请，雇主须提交财务证明（月入不低于港幣15,000元）及声明书。一般4–8周获批。

**转雇**（现有外佣转换雇主）：
- 正常情况下，外佣合约期满或提前终止后方可转雇
- 若因雇主原因（裁员、移民等）终止合同，外佣可申请豁免"两年规定"（即在港居住满2年方可续签的规定），转雇无需离港
- 转雇期间外佣可留港等待新合约入境处批准

## 常见雇主误区与违规风险

| 误区/违规行为 | 实际法律后果 |
|---|---|
| 扣起外佣护照 | 违反《防止酷待家庭佣工条例》，属刑事罪行 |
| 要求外佣在合约地址以外工作 | 违反外佣工作签证条件，雇主可被检控 |
| 不提供住宿或收取住宿费用 | 违反雇佣条例，劳工处可介入调查 |
| 不给足法定假日或强迫在假日工作 | 劳工审裁处可裁定雇主补偿 |
| 解雇怀孕外佣 | 违反《雇佣条例》产假保障条款 |
| 不购买医疗保险 | 违规，且可能导致雇主须自行承担医疗费用 |

香港劳工处设有外佣热线（2717 1771），外佣及雇主均可致电查询，劳工处亦定期主动抽查。合规经营是避免纠纷的最佳策略，在香港雇主方面的法律风险远高于内地的类似雇佣关系。
