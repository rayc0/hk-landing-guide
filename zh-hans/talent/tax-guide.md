# 香港薪俸税指南

香港以低税率著称，是全球税务负担最低的主要金融中心之一。对于内地来港专才，理解香港薪俸税制度不仅有助于准确评估实际税后薪酬，更能通过合法免税和扣减安排，最大化到手收入。

---

## 1. 香港薪俸税税率结构

香港薪俸税采用累进税率，应课税入息实额（扣除免税额后）按以下税阶计算：

| 应课税入息实额（港幣）| 税率 |
|---|---|
| 首50,000元 | 2% |
| 其次50,000元（50,001–100,000）| 6% |
| 其次50,000元（100,001–150,000）| 10% |
| 其次50,000元（150,001–200,000）| 14% |
| 余额 | 17% |

**标准税率**：15%（按应课税入息总额计算，不扣除免税额）

> **关键规则**：税局会分别以累进税率和标准税率计算应缴税款，取**较低者**征税。年薪约港幣150万元以上的高薪人士，通常以标准税率缴税。

## 2. 2025/26年度主要免税额

| 免税额类别 | 金额（港幣） |
|---|---|
| 基本个人免税额 | 132,000 |
| 已婚人士免税额 | 264,000 |
| 子女免税额（每名子女）| 130,000 |
| 供养父母/祖父母免税额（香港同住）| 25,000–50,000 |
| 供养兄弟姊妹免税额 | 37,500 |

---

## 3. 合法扣除项目

除免税额外，以下支出可从应课税入息中扣除：

| 扣除项目 | 上限 |
|---|---|
| 强制性公积金（MPF）强制供款 | 18,000/年 |
| 认可慈善捐款 | 应课税入息35%以内 |
| 个人进修开支 | 100,000/年 |
| 居所贷款利息 | 100,000/年（如自住物业按揭） |
| 长者住宿照顾开支 | 100,000/年 |
| 认可自愿医保计划（VHIS）保费 | 8,000/人/年 |

**MPF自愿供款节税**：自2025年起，「税务可扣减自愿供款」（TVC）的额外供款，每年可获最高港幣60,000元扣减——高薪人士最值得利用的节税工具之一。

---

## 4. 报税时间线

| 阶段 | 时间（每年）|
|---|---|
| 税务局发出报税表 | 5月初 |
| 个人薪俸税报税截止日 | 6月2日（通常） |
| 委托税务代理人申请延期 | 最长可延至11月 |
| 税务局发出税单 | 约10–11月 |
| 第一期税款缴交截止 | 约1月 |
| 第二期税款缴交截止 | 约4月 |

**注意**：首次在港工作的专才，如在港工作少于60天，可豁免薪俸税。此规定对出差频繁或工作开始日期在年中的人士有实际意义。

## 5. 香港 vs 内地税负比较

| 收入水平（年薪港幣等值）| 香港实际税率 | 内地个人所得税税率（参考）|
|---|---|---|
| 50万 | 约6–8% | 约30–35% |
| 100万 | 约9–11% | 约35–40% |
| 200万 | 约11–13% | 约40–45% |
| 500万+ | 约15%（标准税率）| 45%（顶级税率）|

> **实际节税效果**：以年薪港幣100万（税后约港幣89万）为例，相同收入在内地个人所得税负担约为香港的3–4倍。这是香港薪酬即使名义上低于部分内地一线城市（如上海/深圳顶级外资机构），税后实际购买力仍明显占优的核心原因。

## 6. 内地专才常忽略的合法节税安排

**子女免税额**：持依赖人签证的子女在港同住，可申索子女免税额（每名港幣130,000元）。此免税额在首年到港时往往被忽视。

**自愿医保（VHIS）保费扣减**：购买政府认可的VHIS计划，每名受保人每年可扣减最高港幣8,000元保费。家庭成员均可计算。

**个人进修扣减**：在职专业人士参与的CPA、CFA、HKSI等专业资格课程费用，最高可扣减港幣100,000元/年。

**MPF TVC供款**：月薪港幣5万以上的专才，每月额外供款约港幣5,000元至MPF TVC户口，全年可多扣减港幣60,000元，实际节税港幣9,000–10,200元（视税率而定）。

## 7. 常见问题

**Q：我在港工作但部分日子出差内地，是否全部薪金都要缴港税？**
A：香港采用「地域来源原则」——只有源自香港工作的入息才须缴纳薪俸税。如能证明部分工作在港以外进行，可申请按比例豁免，但须保留出行记录和雇主证明。

**Q：雇主提供的住房补贴（Housing Allowance）要交税吗？**
A：是的。雇主提供的住房津贴或住宿安排均须计入应课税入息。但若公司提供「服务式住宿」，计税方式与现金补贴有所不同，建议向税务顾问确认。

**Q：可以聘请会计师代理报税吗？**
A：可以。专业税务代理可协助申请延期报税（最长延至11月）、处理首年报税的过渡安排，以及优化合法扣减。对年薪港幣80万以上的专才，聘请税务代理的成本（通常港幣1,500–5,000元/年）通常物有所值。

---

## 总结

香港薪俸税制度简单、税率低，但并非完全无需规划。个人免税额、子女扣减、MPF TVC供款、VHIS保费、进修费用等合法节税工具，合理运用后可为年薪港幣80万以上的专才节省税款港幣数万元。到港首年建议向税务顾问咨询，尤其在首次报税前，确认地域豁免和过渡安排是否适用。
