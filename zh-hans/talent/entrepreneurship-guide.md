# 内地专才在港创业指南

香港是亚洲领先的创业和商业枢纽，拥有简便的公司注册制度、低税率环境、国际融资渠道和大湾区腹地支持。对持有IANG、优才或高才通签证的内地专才，香港提供了独特的创业窗口——在保留香港居留权的同时，将业务延伸至国际和内地市场。

---

## 1. 香港创业生态概况

| 维度 | 香港的优势 |
|---|---|
| 公司注册 | 全球最快之一：网上注册最快1个工作日，政府费用约港幣1,720元 |
| 税务 | 利得税16.5%（首200万利润优惠税率8.25%）；无增值税、无股息税 |
| 法律体系 | 普通法体系；合同执行力强；知识产权保护国际标准 |
| 融资渠道 | 联交所（香港交易所）；国际PE/VC；家族办公室；政府资助 |
| 大湾区腹地 | 毗邻深圳；CEPA优惠；前海合作区；河套创新园 |
| 人才 | 全球招聘无障碍；80+国家无需签证进港 |

## 2. 内地专才的创业签证路径

| 签证类型 | 适用创业者 | 主要条件 |
|---|---|---|
| **IANG（输入内地人才）自雇** | 曾持IANG工签、转为自雇 | 须证明有合理业务计划和资金来源；移民局批准 |
| **优才计划（QMAS）** | 综合计分制下以"创业"获批 | 创业能力和成就列入评分；批准后可自雇 |
| **高才通（TTPS）** | A类/B类人才在港自主创业 | 批准后前2年无需雇主担保；可自雇 |
| **投资者签证（废止已久）** | 已停止受理 | — |

> **现实提示**：高才通（TTPS）A类/B类人才获批后，在港2年内享有最高灵活度——可加入初创公司（含持股）、自行注册公司、或以个人顾问名义接单。这是目前内地专才在港创业最常见的路径。

## 3. 在港注册公司（有限公司）

**流程概述**：
1. 通过公司注册处（Companies Registry）网上平台申请
2. 选择公司名称（中文名+英文名，须唯一）
3. 提交章程细则（Articles of Association）
4. 注册资本最低港幣1元（建议港幣10,000元）
5. 领取商业登记证（Business Registration Certificate）

**费用参考**：

| 项目 | 费用（港幣）|
|---|---|
| 公司注册费 | 1,720 |
| 商业登记证（首年）| 2,000 |
| 法定秘书服务（首年）| 2,000–5,000 |
| 注册地址（虚拟办公室）| 2,400–6,000/年 |
| **初始总成本** | **约8,000–15,000** |

## 4. 政府创业支持计划

| 计划 | 机构 | 资助额 | 适合阶段 |
|---|---|---|---|
| 数码港培育计划 | 数码港（Cyberport） | 港幣500,000 | 科技/Fintech早期 |
| HKSTP（科学园）孵化 | 香港科技园 | 港幣490,000–港幣1,300,000 | 硬科技/生物科技 |
| 创科创投基金（ITVF）| 创新科技署 | 按1:2配比投资 | 需先获私人VC投资 |
| 科技券计划（TVP）| 创新科技署 | 最高港幣600,000 | 中小企科技升级 |
| BUD专项基金 | 香港生产力促进局 | 最高港幣700,000 | 拓展内地/东盟市场 |

> **申请现实**：Cyberport和HKSTP的培育计划竞争激烈，录取率约10–15%。申请须在香港注册、有明确商业模式和本地运营计划。内地专才在创始团队中须有本地居留权，或以香港为主要运营地。

## 5. 香港初创融资生态

| 融资渠道 | 特点 | 适合阶段 |
|---|---|---|
| 天使投资（本地HNW/Family Office）| 投资金额港幣50–500万；决策快 | 种子轮 |
| 本地VC（Gobi/Vectr/Brinc等）| 专注大湾区和东南亚市场 | A轮前 |
| 国际VC（Sequoia/Softbank等）| 票面大；条件严格；需要规模潜力 | A轮及以后 |
| 政策性资金（ITVF/HKSTP）| 非稀释性资助+配比投资 | 早期验证阶段 |
| 香港联交所（上市）| GEM（创业板）vs 主板；监管严格 | 成熟阶段 |

**与内地VC的主要区别**：
- 香港VC普遍使用国际标准Term Sheet（SAFE/KISS/优先股）
- 条款更保护投资人；创业者须熟悉清算优先权（Liquidation Preference）
- GP关系文化重要，但比内地更依赖数据和指标

## 6. 粤港澳大湾区创业优势

| 大湾区节点 | 对港创业者的价值 |
|---|---|
| 深圳前海合作区 | 内地企业所得税15%；香港专才可申请内地社保豁免 |
| 河套深港科技创新合作区 | 两地监管协同；生物科技特殊豁免 |
| 横琴粤澳深度合作区 | 更接近澳门；财富管理和金融科技聚焦 |
| CEPA优惠 | 香港注册公司在内地业务享关税和准入优惠 |

## 7. 签证续签：创业者的注意事项

在港创业的内地专才，签证续签须证明业务真实运营：

- 提供公司财务报表（即使亏损也需如实提交）
- 证明业务合理性：客户合同、银行流水、纳税证明
- 建议保持最低港幣50,000元公司银行账户余额
- 若业务失败，可凭新雇主续签IANG工签——创业不成功不等于自动失去留港资格

---

## 总结

香港创业生态对内地专才既开放又有现实门槛。高才通是目前最灵活的创业路径——前2年无需雇主担保，可以自由探索商业模式。政府资助（Cyberport、HKSTP）提供了非稀释性资金，但竞争激烈、条件严格。大湾区腹地是香港创业者独有的战略优势：在国际规则下运营香港公司，同时深度利用内地市场和供应链，是当前最具竞争力的组合之一。
