# 香港の保険ガイド

香港はアジア有数の保険センターであり、多種多様な保険商品が揃っています。日本からの駐在員・移住者にとって最優先の課題は医療保険の確保ですが、住宅保険・生命保険・MPF（強制積立年金）の仕組みも把握しておく必要があります。

---

## 1. 香港の医療制度と保険の必要性

香港の医療体制は「公立」と「民間」の二本立てです。

| 比較項目 | 公立病院（Hospital Authority） | 民間病院・クリニック |
|---|---|---|
| 費用 | 香港居民は非常に安い（入院HKD 120/日） | 外来HKD 500〜2,000/回；入院HKD 5,000〜/日 |
| 待機時間 | 救急以外は数週間〜数ヶ月待ち | 予約当日または翌日が多い |
| 言語 | 広東語が主（英語対応は限定的） | 英語・日本語対応クリニックあり |
| 適合シーン | 慢性疾患・入院治療 | 急な発熱・外来診察 |

就労ビザで来港した日本人は公立病院を利用できますが、待機時間と言語の壁から、民間医療保険への加入が実質的に必要です。

## 2. 医療保険の種類比較

### グループ保険（雇用主提供）

雇用主が提供する団体保険は最もコストパフォーマンスが高い選択肢です。

- **一般的なカバー範囲**：外来受診（HKD 250〜500/回）、入院・手術
- **家族カバー**：配偶者・子どもは別途追加が必要な場合が多い
- **注意点**：指定医療機関ネットワーク内での受診に限られる場合あり

### 個人医療保険

グループ保険で家族がカバーされない場合や、補完目的で個人加入します。

### VHIS（自発的医療保険計画）指定プラン

政府が認定する医療保険スキームで、税控除の対象となります。

| 特徴 | 内容 |
|---|---|
| 監督機関 | 保険業監管局（IA）認定 |
| 更新保証 | 保険会社は健康状態に関わらず更新を拒否できない |
| 基本カバー | 入院・手術・日帰り手術・精神科治療（標準プランで23カテゴリー） |
| 税控除額 | 被保険者1名につき最大HKD 8,000/年 |
| 主要プロバイダー | AIA、AXA、BUPA、Blue Cross、Cigna、Sun Life |
| 目安月額保険料 | 成人HKD 300〜800；子ども HKD 200〜500 |

**主要プロバイダー比較**：

| プロバイダー | 特徴 | おすすめの人 |
|---|---|---|
| AIA（友邦保険） | 総合カバー豊富；ネットワーク広い | 家族での包括加入 |
| AXA（安盛） | 国際ネットワーク強；出張対応 | 頻繁に出張する人 |
| BUPA | 香港での実績長い；サービス評判良 | 長期香港在住者 |
| Blue Cross | コストパフォーマンス高い | 予算を抑えたい人 |
| Cigna | 国際カバー充実 | グローバルに生活する人 |

## 3. VHISの税控除の受け方

VHISの税控除は薪俸税（サラリーマン税）申告時に適用されます。

1. VHIS指定プランの保険証書と支払い領収書を保管する
2. 年末に保険会社が発行する「税務証明書」を受け取る
3. 税務申告書（BIR60）の「VHIS保険料控除」欄に記入する
4. 最大控除額：被保険者1名あたりHKD 8,000（自分・配偶者・子ども各人に適用可）

**日本の健康保険との関係**：日本の国民健康保険・社会保険は、原則として海外居住者は脱退または任意継続が可能です。詳細は後述のFAQを参照。

## 4. 住宅保険（家財保険・第三者賠償責任）

香港の多くの賃借人は住宅保険に未加入ですが、比較的安価で実用的な保障です。

| 保険種別 | 主なカバー内容 | 年額目安 |
|---|---|---|
| 賃借人向け家財保険 | 家具・家電・衣類等の損害；第三者賠償責任 | HKD 500〜2,000/年 |
| 所有者向け住宅保険 | 建物構造・内装・所有者賠償責任 | HKD 2,000〜5,000/年 |

**月額換算**：賃借人向けはHKD 100〜200/月程度と非常に安価です。

**重要ポイント**：
- 賃貸契約は通常、入居者が人的損害（水漏れ・火災等）に対して責任を負う条項を含む
- 建物管理組合の共有部分保険は専有部分の損害をカバーしない
- 日本と異なり、家財保険は火災だけでなく盗難・水害等もカバーすることが多い

## 5. 生命保険：香港ポリシーの特徴と日本との違い

| 比較項目 | 香港の生命保険 | 日本の生命保険 |
|---|---|---|
| 為替 | 主にUSD建てまたはHKD建て | JPY建てが主流 |
| 税制環境 | 低税率；相続税なし | 相続税・贈与税あり |
| 定期保険 | 純粋な死亡保障；月額安い | 同様の商品あり |
| 貯蓄型保険 | USD建て複利；高純資産層に人気 | 予定利率は近年低水準 |
| 解約ペナルティ | 10〜25年ロックイン期間が一般的 | 同様のリスクあり |

**香港ポリシーの注意事項**：
- 保険契約は香港法に基づき管轄される
- 解約返戻金は長期保有することで増加するが、早期解約は大きな損失となる
- 購入前に必ず独立FP（ファイナンシャルプランナー）への相談を推奨

## 6. MPF（強制積立年金）の仕組みと運用

香港で就労するすべての従業員（18〜65歳）はMPFへの加入が義務付けられています。

| 項目 | 内容 |
|---|---|
| 強制拠出率 | 雇用主・従業員それぞれ月給の5% |
| 拠出上限 | 月給HKD 30,000が上限（最大HKD 1,500/月/人） |
| ファンド選択 | 通常10〜15種類（株式型・債券型・保守型・マネーマーケット型） |
| ポータビリティ | 転職時に新雇用主のMPF口座または個人口座に移管可 |
| 引き出し時期 | 原則65歳（香港永久離脱・重度障害等の特例あり） |

**ファンド選択のポイント**：
- 退職まで20年以上ある場合：株式型ファンドへの配分比率を高める（成長重視）
- 退職10年以内：債券型・保守型へ段階的にシフト（資産保全）
- 年1回ファンド変更の確認を推奨（同一プラン内での変更は無料）

**税控除自発拠出（TVC）**：年間最大HKD 60,000をMPF TVC口座に追加拠出でき、全額が薪俸税の課税対象所得から控除されます。

## 7. よくある質問

**Q: 日本の健康保険は解約すべきですか？**
就労ビザで香港に移住した場合、日本の勤務先の社会保険（健康保険・厚生年金）は通常、退職に伴い自動的に脱退となります。フリーランスや自営業の場合、日本の国民健康保険は海外転出届提出後に解約できますが、任意継続（最大2年）も可能です。香港居住中は現地医療保険で対応するのが一般的です。

**Q: VHISプランは年齢が上がると保険料が大幅に上がりますか？**
VHISは年齢帯別に保険料が設定されており、加齢とともに上昇します。ただし、VHISの規定上、更新を拒否または既往症を理由に保険料を引き上げることは禁じられています（標準プランの場合）。

**Q: 会社のグループ保険だけで十分ですか？**
外来・入院の基本カバーはグループ保険で対応できますが、家族（配偶者・子ども）がカバーされていない場合や、高度な医療・専門医受診には補完的な個人保険（VHIS）の追加が推奨されます。

**Q: 転職時にMPFはどうなりますか？**
転職時、従業員の強制拠出分はすべて自分のものです。雇用主拠出分は勤続年数に応じて権利確定（Vesting）します（多くのプランでは1年以上で一部、2年で全額）。新雇用主のプランに移管するか、個人口座（Personal Account）を開設して管理できます。
