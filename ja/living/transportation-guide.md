# 香港交通ガイド

香港の公共交通網は世界有数の効率性を誇り、香港島から新界まで地下鉄・バス・フェリーが高密度に整備されています。日本と同様に交通機関への依存度が高い都市ですが、オクトパスカードの使いやすさや運賃の安さでは香港が優位で、ほぼすべての移動を公共交通でカバーできます。

---

## 1. MTR（Mass Transit Railway）— 香港の地下鉄

香港の地下鉄は現在9本の路線で構成されており、市街地のほぼすべてをカバーしています：

| 路線 | カバーエリア | 主要駅 |
|---|---|---|
| 荃湾線 | 中環〜荃湾 | 中環・銅鑼灣・旺角・荃湾 |
| 港島線 | 堅尼地城〜柴灣 | 香港大學・上環・金鐘・銅鑼灣 |
| 観塘線 | 油麻地〜調景嶺 | 旺角・九龍湾・観塘 |
| 将軍澳線 | 北角〜康城 | 油塘・寶琳・將軍澳 |
| 東涌線 | 香港（中環）〜東涌 | 機場鐵路と共用・青衣・東涌 |
| 西鐵線 | 屯門〜紅磡 | 元朗・天水圍・九龍・紅磡 |
| 馬鞍山線 | 大圍〜烏溪沙 | 沙田・大圍・馬鞍山 |
| 迪士尼線 | 欣澳〜迪士尼 | 迪士尼リゾート専用 |
| 機場快綫 | 香港〜機場 | 九龍・青衣・空港 |

**通勤の実用テクニック**：
- 各駅のホームにある「乗車位置マーク」（黄色のステッカー）に従って並ぶのが慣例
- ラッシュアワーは月〜金 7:30〜9:30 / 17:30〜19:30
- 早朝割引（MTR Early Bird）：特定駅では7:45以前の乗車で運賃割引

## 2. オクトパスカード（八達通）

香港の交通系ICカードで、地下鉄・バス・フェリーはもちろん、コンビニ・スーパーでも使えます：

| カード種類 | 対象 | デポジット |
|---|---|---|
| 通常オクトパス | 12〜64歳 | HKD 50 |
| 子供用 | 3〜11歳 | HKD 9 |
| シニア用 | 65歳以上 | HKD 9 |
| 旅行者用（Tourist Octopus）| 観光客 | デポジット不要（1日/2日券）|

**チャージ方法**：MTR各駅の自動チャージ機（現金/クレカ）、コンビニ（7-Eleven/サークルK）、オクトパス公式アプリ（一部カード）

**月定期（Monthly Pass）**：MTRの指定区間で毎月HKD 250〜500程度の定額パス。通勤経路が固定している場合は約20〜30%の節約効果。

## 3. バスとミニバス

| 種類 | 特徴 | 運賃 |
|---|---|---|
| フランチャイズバス（KMB/城巴等）| 主要幹線；エアコン完備；オクトパス可 | HKD 3〜15 |
| 緑ミニバス（Green Minibus）| 固定路線・停車所；14〜16人乗り | HKD 3〜10 |
| 赤ミニバス（Red Minibus）| 停車場所が柔軟；現金またはオクトパス | HKD 3〜15（要交渉の場合も）|

**バス到着情報のアプリ**：KMBアプリ・バス到着時刻リアルタイム表示。「次のバスはいつ？」は「Next Bus Arrives in X minutes」の電光掲示板で確認可能。

## 4. タクシー

香港のタクシーは車体の色でエリアを識別できます：

| 色 | 運行エリア | 初乗り料金 |
|---|---|---|
| 赤（市区タクシー）| 香港全土（市街地）| HKD 30 |
| 緑（新界タクシー）| 新界エリア内のみ | HKD 23.5 |
| 青（大嶼山タクシー）| 大嶼山のみ | HKD 22 |

**料金体系**（赤タクシーの場合）：初乗り2km HKD 30、以降200m/1分ごとにHKD 1.9加算。荷物料金（HKD 6/個）、空港サービス（HKD 5加算）あり。

**タクシーアプリ**：HK Taxi、Grab（香港対応）が主要。Uberは制限あり。

## 5. フェリー（渡輪）

| 路線 | 所要時間 | 運賃参考 |
|---|---|---|
| 中環〜ツィムサーチョイ（天星小輪）| 約8分 | HKD 3.4（下层）|
| 中環〜ランタオ島（東涌/大澳）| 35〜55分 | HKD 17〜30 |
| 中環〜南Y島（榕樹灣/索罟灣）| 25〜30分 | HKD 20〜30 |
| 中環〜長洲島 | 35〜55分 | HKD 19〜34 |

天星小輪（Star Ferry）はビクトリア湾を渡る観光名所でもありますが、HKD 3.4という破格の運賃で通勤にも使われています。

## 6. 高速鉄道（高鐵）西九龍駅

中国本土への最速アクセス：

| 行き先 | 所要時間 | 頻度 |
|---|---|---|
| 深圳北 | 約19分 | 15分〜20分に1本 |
| 広州南 | 約48分 | 1時間に数本 |
| 北京西 | 約8時間15分 | 数本/日 |
| 上海虹橋 | 約8時間55分 | 数本/日 |

**注意**：西九龍駅では「一地両検」方式を採用。香港側で中国出境・入境手続きをまとめて行うため、駅到着は出発45〜60分前が目安。

## 7. 月平均交通費の試算

| 通勤パターン | 月間費用目安 |
|---|---|
| MTR通勤（市内短距離）| HKD 400〜600 |
| MTR通勤（新界〜市区・長距離）| HKD 700〜1,200 |
| バス中心の通勤 | HKD 300〜500 |
| MTR + バス + タクシー（混合）| HKD 800〜1,500 |

> MTR駅近の住宅は家賃が15〜25%高い場合が多いですが、通勤時間の短縮と日常の利便性がそれを補うケースが多いです。

---

## まとめ

香港の公共交通はMTRを中心に発達しており、オクトパスカードがあれば地下鉄・バス・フェリー・コンビニまで1枚でカバーできます。タクシーは日本より安く、アプリ配車も充実しています。新界〜市区の長距離通勤者には月定期パスが有効です。
