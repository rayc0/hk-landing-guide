# 香港科技園（HKSTP）・数码港インキュベーション完全ガイド：スタートアップへの補助金・資源

**概要：** 香港には世界でも有数のスタートアップ支援インフラが整っています。その中核を担うのが**香港科技園公司（HKSTP）**と**香港数码港（Cyberport）**です。両機関はそれぞれ異なる技術分野のスタートアップを対象に、キャッシュ補助・オフィス施設・メンタリング・投資家ネットワークを提供するインキュベーションプログラムを運営しています。本記事では、日本人創業者および日本企業の香港スタートアップが、この2つのプログラムをどのように活用できるかを徹底解説します。

---

## 一、HKSTP vs. Cyberport：どちらを選ぶか？

まず両機関の全体像を比較します。

| 比較項目 | 香港科技園（HKSTP） | 香港数码港（Cyberport） |
|---|---|---|
| 設立年 | 2001年 | 2004年 |
| 主な対象分野 | ハードウェア科技、生物科技（バイオテック）、高度製造、AI・ロボティクス、グリーンテック | フィンテック、サイバーセキュリティ、ビッグデータ、スマートシティ、デジタルエンタテインメント |
| インキュベーションプログラム | INCU-Bio / INCU-App / INCU-Tech / INCU-HK | Cyberport Incubation Programme（CIP）/ Cyberport Creative Micro Fund（CCMF） |
| 最大補助額 | **HKD 130万**（INCU-Tech） | **HKD 50万**（CIP） |
| 拠点 | 沙田科学園（香港新界） | 鋼綫灣（香港島） |
| 入居企業数 | 約600社以上 | 約1,900社以上（エコシステム全体） |
| 採択率 | 約10〜15% | 約15〜20% |
| 主な投資家ネットワーク | VC、製造業パートナー、大学研究機関 | フィンテックVC、銀行パートナー、デジタルメディア企業 |

**簡単な指針：**
- **物理的な製品を開発している場合（ハードウェア、医療機器、バイオ）→ HKSTP**
- **ソフトウェア、フィンテック、デジタルサービスを開発している場合 → Cyberport**

## 二、HKSTPインキュベーションプログラムの詳細

HKSTPのインキュベーション（孵化）プログラムは複数のトラックに分かれており、創業段階と技術分野によって適切なプログラムが異なります。

### 2.1 各プログラムの比較

| プログラム | 対象分野 | インキュベーション期間 | 最大補助額 | 主なサポート |
|---|---|---|---|---|
| **INCU-Bio** | 生物科技、医療機器、ヘルステック | 最大5年 | **HKD 130万** | ウェットラボ施設、臨床試験ネットワーク |
| **INCU-Tech** | AI・ロボティクス、先進製造、電子機器、グリーンテック | 最大3年 | **HKD 130万** | プロトタイプラボ、製造業パートナー紹介 |
| **INCU-App** | モバイルアプリ、クラウドサービス、SaaS | 最大2年 | **HKD 38万** | コワーキングスペース、クライアント紹介 |
| **INCU-HK** | 香港大学発スピンオフ企業 | 最大3年 | **HKD 130万** | 大学研究リソース、教授メンタリング |

### 2.2 INCU-Tech／INCU-Bioの段階別補助額

補助金は一括交付ではなく、マイルストーン達成に応じた段階的な交付となります：

| ステージ | 条件 | 補助額の目安 |
|---|---|---|
| 参加時補助 | プログラム参加契約締結 | HKD 15〜30万 |
| 中間補助 | 技術実証・プロトタイプ完成 | HKD 30〜50万 |
| 最終補助 | 製品ローンチ・初期顧客獲得 | HKD 30〜50万 |
| 合計最大 | 全マイルストーン達成 | **HKD 130万** |

## 三、Cyberportインキュベーションプログラムの詳細

### 3.1 Cyberport Incubation Programme（CIP）

| 項目 | 内容 |
|---|---|
| 期間 | 最大2年 |
| 最大補助額 | **HKD 50万** |
| 補助の内訳 | 現金補助 HKD 30万 ＋ オフィス賃貸補助 HKD 20万 |
| 主な対象 | フィンテック、AI、ビッグデータ、デジタルエンタテインメント、スマートシティ |
| 採択数 | 年間約60〜80社 |

### 3.2 Cyberport Creative Micro Fund（CCMF）

インキュベーションプログラムに入る前段階として、**CCMF（クリエイティブ・マイクロファンド）**があります。

| 項目 | 内容 |
|---|---|
| 補助額 | **HKD 10万**（上限） |
| 対象 | アイデア段階・プロトタイプ開発段階のプレシード企業 |
| 審査 | ピッチ方式（書類審査不要） |
| 利点 | CIPへの優先審査パスとして機能することが多い |

**戦略的活用：** CCMFで最初のHKD 10万を取得し、プロトタイプを完成させた後にCIPへ申請するという2段階アプローチが、Cyberportで成功している日本系スタートアップの典型パターンです。

## 四、申請要件と選考基準

### 4.1 HKSTP申請要件

- **技術の革新性**：既存市場への模倣ではなく、独自技術・知的財産（IP）を有すること
- **チームの実力**：技術系創業者と事業系創業者のバランス——HKSTP は「チーム」を重視
- **市場規模**：対象市場がグローバル規模（香港市場のみでは不可）であること
- **香港との関連性**：香港での技術開発・事業拠点を設ける意志と計画

### 4.2 Cyberport申請要件

- **デジタル技術の活用**：サービスの核にデジタル技術があること
- **スケーラビリティ**：スケールアップが可能なビジネスモデル
- **チームの多様性**：フィンテックの場合、金融業界経験者の参加が高評価
- **社会的インパクト**：スマートシティ・ESG関連のインパクト訴求が採択率を上げる傾向

### 4.3 採択率について

採択率は公式には公表されていませんが、業界の共通認識は以下の通りです：

| 機関 | 推定採択率 |
|---|---|
| HKSTP（INCU-Tech/Bio） | **10〜15%** |
| Cyberport CIP | **15〜20%** |
| Cyberport CCMF | **25〜35%** |

競争率は高いですが、**日本語・英語・広東語が堪能な多言語チーム**や、**日本の大手企業との既存の技術開発関係**を持つ申請者は差別化要素として評価される傾向があります。

## 五、インキュベーション期間のサポート内容

### 5.1 物理的インフラ

**HKSTP：**
- 研究開発ラボ（クリーンルーム、生物安全実験室、プロトタイプ工房）
- 入居企業専用コワーキングスペース
- 各種検査・測定機器の共用利用

**Cyberport：**
- スタートアップ専用コワーキング「SmartSpace」
- 最新のデジタルメディア・VR/AR制作スタジオ
- サイバーセキュリティテストベッド

### 5.2 メンタリングとネットワーク

| サポートの種類 | HKSTP | Cyberport |
|---|---|---|
| メンタリング | 技術専門家・業界OBによる定期セッション | フィンテック・投資銀行業界メンターネットワーク |
| 投資家紹介 | HKSTP Venture Programme（内部VC）経由 | 投資家マッチングイベント「Cyberport Investors Network」 |
| パイロット顧客 | 製造業・ヘルスケア系大企業との連携機会 | HSBC、Standard Chartered等の銀行パートナーとのPoC機会 |
| 政府関係者 | InnovatechFund、Science Park系補助金への優先ルート | フィンテック規制サンドボックス（金管局）への紹介パス |

### 5.3 卒業後の資金調達実績

HKSTP・Cyberport卒業生は、プログラム在籍中および卒業後2年以内に以下のような資金調達を実現しています（公開情報ベース）：

- HKSTPインキュベーション卒業生の**約30〜40%**がシリーズA以上を調達
- Cyberport卒業生の累積スタートアップ評価額は**HKD 300億超**（2025年時点）

## 六、日本人創業者向けアドバイス

### 6.1 日本人が持ちやすい強みと注意点

**強み：**
- 日本の高品質製造技術・精密機械分野の深い専門知識（HKSTP INCU-Techで高評価）
- 日本の大手企業・病院・官公庁との既存ネットワーク（パイロット顧客としての説得力）
- 日本のデザイン・UX感覚（Cyberportのデジタルエンタテインメント枠で評価されることも）

**注意点：**
- **英語・広東語の申請書が必須**——日本語のみの資料は評価対象外
- **チームに非日本人メンバーを含めることを推奨**——多様性が採択評価を上げる
- **香港市場だけを対象としないビジョン**が重要——「グローバル展開の足がかりとして香港を選んだ」という戦略的理由が求められる

### 6.2 申請準備のタイムライン目安

| 時期 | アクション |
|---|---|
| 申請3ヶ月前 | 技術デモ資料・財務モデルの準備開始 |
| 申請2ヶ月前 | Cyberportの場合はCCMFから先に申請（PoC資金確保） |
| 申請1ヶ月前 | 応募書類（英語）の最終仕上げ・メンターレビュー依頼 |
| 申請時 | オンライン申請フォーム提出 ＋ ピッチデック添付 |
| 申請後2〜4週間 | 面接・ピッチセッション招待（通過者のみ） |
| 採否通知 | 申請から約6〜8週間後 |

### 6.3 HKSTPとCyberportへの同時申請は可能か？

**可能ですが、推奨されません。** 両機関とも「本気のコミットメント」を求めており、同時申請は「どちらにでも入りたい」という印象を与えリスクがあります。技術分野に応じて優先順位をつけてから申請することを強くお勧めします。

## 七、スタートアップビザとの組み合わせ

HKSTPまたはCyberportのインキュベーションプログラムに採択された場合、香港入境事務処への**スタートアップビザ（就業ビザ）申請が大幅に有利**になります。

- 採択通知書がビザ申請の「実質的な雇用証明」として機能
- 入境事務処は両機関の採択を高く評価（却下率が大幅に低下）
- 家族（配偶者・未成年の子ども）の帯同ビザ申請も容易に
