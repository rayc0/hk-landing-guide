# 香港の外国人家庭内ヘルパー（FDH）ガイド：雇用法・費用・採用チャネル

香港では約10万世帯以上が外国人家庭内ヘルパー（Foreign Domestic Helper、以下FDH）を雇用しており、これは全世帯の約14%に相当する。子育て・家事・高齢者ケアの担い手として、香港の共働き家族にとってヘルパー雇用は「特別なこと」ではなく「当たり前の選択」となっている。日本人家族がこの制度をうまく活用するためには、法的条件と雇用後の関係構築の両方を理解することが重要だ。

## ヘルパー制度の概要

香港のFDH制度は労工処（Labour Department）が管理し、雇用条件・最低賃金・住居要件・ビザ手続きまでを一元的に規定している。主な出身国はフィリピン（約55%）とインドネシア（約40%）で、英語能力・家事スキル・子育て経験においてそれぞれ特徴がある。

**フィリピン人ヘルパーの特徴**：英語堪能（フィリピン国内の公教育が英語）、子供への教育的関与（読み聞かせ・英語会話）に強み。

**インドネシア人ヘルパーの特徴**：家事全般（料理・清掃）に丁寧で従順な傾向。マレー語圏のため広東語習得が早い場合も。

## 法定雇用条件一覧

| 条件 | 内容 |
|---|---|
| 最低賃金（2024年） | HKD 4,990/月 |
| 食費手当 | HKD 1,196/月（現金または同等の食事提供） |
| 雇用契約期間 | 2年間（標準雇用契約書に基づく） |
| 住居 | 雇用家族の住居内に無料で提供（独立した部屋が理想） |
| 週休 | 週1日の法定休日（通常日曜日） |
| 有給休暇 | 1年目：7日間、以降年数に応じて増加 |
| 年間法定祭日 | 12日間 |
| 医療保険 | 雇用主が全額負担の医療保険加入が義務 |
| 帰国航空券 | 契約終了時に雇用主が提供義務 |

**レビー税（Levy）**：雇用主はHKD 400/月の外国人家庭内ヘルパー雇用税を政府に納付する義務がある。

## 採用方法の比較

| 採用方法 | メリット | デメリット | 費用感 |
|---|---|---|---|
| エージェント経由 | 書類手続きサポート・事前面接・保証あり | 手数料高め（HKD 5,000〜15,000） | 高 |
| オンライン（Helperplace/HelperChoice等） | 直接コンタクト可能・選択肢が多い | 書類手続きは自己管理 | 中 |
| 知人紹介 | 信頼度が高い・定着率が良い | 候補者が限られる | 低 |
| 在港ヘルパーの転職（Transfer） | 即戦力・香港経験あり・面接可能 | 前雇用主トラブルの確認が必要 | 低〜中 |

**「転職（Transfer）ヘルパー」とは**：現在香港に滞在中で、前雇用主との契約が終了（または途中終了）したヘルパーのこと。来港後すぐに働けるため即戦力だが、前職の離職理由の確認が重要。

## 雇用後の手続き

1. **労工処への契約登録**：標準雇用契約書（ID407）を3部作成し、双方署名後に労工処へ提出。
2. **ビザ申請（新規来港の場合）**：入境事務処へ就業ビザを申請。通常4〜8週間かかる。
3. **医療保険加入**：最低限度の入院・通院保険を雇用主名義で加入。
4. **住居の準備**：独立した寝室の提供が望ましい（法的には「適切な宿泊場所」を提供すれば足りるが、プライバシー確保が定着率に直結）。

## 日本人家族でのヘルパー活用法

日本人がヘルパー雇用で最初に戸惑うのは「他人に家の中を任せること」への文化的抵抗感だ。以下の点を意識すると関係構築がスムーズになる。

**業務範囲の明確化**：採用前に「家事の優先順位」「調理の希望スタイル（日本食の対応可否）」「子供への接し方のルール」を書面で共有すると認識のズレが減る。

**コミュニケーション言語**：英語が共通言語になるケースが多い。シンプルな英語で具体的に指示することが重要。日本語でのコミュニケーションを期待するのは現実的ではないが、日常フレーズを少しずつ教えていく家族も多い。

**相互尊重の文化**：香港ではヘルパーに対して尊重ある態度を持つことが定着率に直結する。誕生日のお祝い、休日の配慮、体調時のサポートなど、小さな気遣いが長期的な信頼関係を生む。

## よくある問題と解決策

**Q: ヘルパーが病気になった場合はどうなりますか？**
A: 雇用主が提供する医療保険が適用される。通常のGP受診費用は保険でカバーされる。重篤な場合は一時帰国が必要になる場合もあり、その際の費用分担は契約書で事前確認が重要。

**Q: ヘルパーと合わなかった場合、契約を途中解除できますか？**
A: 双方合意のうえで途中解除は可能だが、雇用主側の一方的解除の場合は1ヶ月分の給与（または1ヶ月前の予告）と帰国航空券の提供が必要。

**Q: ヘルパーに日本食を調理してもらえますか？**
A: 可能だが、採用時に確認が必要。フィリピン人・インドネシア人ともに、YouTubeやレシピを活用しながら日本食に対応できるヘルパーは多い。面接時に調理スキルと習得意欲を確認しよう。

**Q: 子供がヘルパーに懐きすぎることはありませんか？**
A: 自然な愛着は問題ないが、親の育児方針をヘルパーと明確に共有することで、教育・しつけ面での一貫性を保てる。定期的な「家族ミーティング」でフィードバックを共有することを推奨する。
