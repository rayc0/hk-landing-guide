# 香港の医療制度：公的医療・私立病院・日本人向け医療機関

## 香港の二層医療システム

香港の医療制度は、公立と私立の二層構造で成り立っている。

**公立医療（HA：Hospital Authority管轄）** は政府が高度に補助しており、香港居民は極めて低コストで医療サービスを受けられる。一方で、需要超過による長い待機時間が慢性的な課題となっている。

**私立医療セクター** は市場原理で運営され、待機なく高品質のサービスを受けられる代わりに、費用は全額自己負担となる。駐在員・外国人居住者の多くは私立医療を利用し、その費用を会社支給の医療保険でカバーするモデルが標準となっている。

## 公立医療と日本の医療の違い

| 比較項目 | 香港公立 | 日本国民健康保険 |
|---|---|---|
| 費用（外来） | HKD 180/回（標準） | 自己負担3割 |
| 費用（救急） | HKD 1,230（2024年） | 自己負担3割＋救急搬送費 |
| 入院費（1日） | HKD 120（標準病棟） | 自己負担3割＋食事代 |
| 待機時間（一般外来） | 数時間〜数日 | 通常当日〜翌日 |
| 言語対応 | 広東語中心（英語対応あり） | 日本語 |
| 専門医アクセス | 紹介状経由・長待機 | 直接受診可（一部紹介状推奨） |
| 薬の提供 | 外来で直接処方 | 院外薬局 |

香港公立病院は技術水準は高いものの、一般外来の待機は半日〜1日以上が当たり前。緊急でない症状での公立病院利用は事実上困難であり、多くの駐在員は日常的な受診には私立クリニックを利用する。

外国人・非永久居民の公立医療利用は可能だが、永久居民より高い料金が適用される（約8倍程度）。

## 私立医療の費用感

私立医療の費用は日本と比較して高額になりやすい。以下は目安：

| 受診種別 | 費用目安（HKD） |
|---|---|
| 一般科（GP）外来 | 400〜800 |
| 専門医外来（内科・外科等） | 1,000〜2,500 |
| 緊急外来（A&E） | 2,000〜5,000 |
| 入院（1日、一般病棟） | 5,000〜12,000 |
| 入院（1日、個室・私立病院） | 15,000〜30,000 |
| 手術（軽度） | 30,000〜80,000 |
| 出産（私立） | 80,000〜200,000 |
| MRI検査 | 5,000〜12,000 |
| 血液検査パネル | 800〜3,000 |

主要私立病院としては、香港養和醫院（Matilda/Hong Kong Adventist）、聖保祿醫院（St. Paul's Hospital）、嘉諾撒醫院（Canossa Hospital）、港怡醫院（HKSH）などが代表的。

## 医療保険の必要性

香港で生活する外国人にとって、**充実した医療保険は必須**である。日本の国民健康保険は香港では使用できず、帰国後の治療にしか適用されない。

一般的な駐在員向け医療保険の保障内容として求められるものは：
- 入院費フルカバー（上限HKD 100万以上/年）
- 外来診療カバー（GP・専門医）
- 救急医療・救急搬送
- 年間健康診断
- 歯科・眼科（オプション）

日系企業の多くは会社として団体医療保険を提供しているが、保障上限や対象施設に制限がある場合もある。特に出産・重大疾患に備え、個人補完保険の加入を検討することが推奨される。

## 日本語対応医療機関

香港では日本語対応の医療機関が複数存在し、日本人コミュニティに広く利用されている。

| 機関名 | 地区 | 特徴 |
|---|---|---|
| 日本語救急・専門医ライン | 各私立病院 | 通訳サービス（事前確認要） |
| 聖保祿醫院（St. Paul's Hospital） | 銅鑼灣 | 日本語通訳スタッフ在籍、日本人医師在籍ケースあり |
| マチルダ国際病院（Matilda International） | 山頂 | 国際患者対応、英語中心・一部日本語可 |
| JAL系列クリニック（Japan Airlines Medical Clinic） | 中環・銅鑼灣 | 日系企業駐在員向け、日本語対応 |
| 日本語対応GP（個人クリニック） | 銅鑼灣・中環・九龍城 | 日本人医師または日本語通訳スタッフ在籍 |
| 日本人メディカルセンター（JMHK） | 中環 | 日本語医療相談窓口 |

最新の日本語対応クリニック情報は、香港日本人商工会議所（JCCI）または在香港日本国総領事館の案内ページで確認することを推奨する。

## 薬局・処方箋システム

香港の薬局は大きく2種類に分かれる：

- **登録薬剤師がいる薬局（Registered Pharmacy）**：処方薬の調剤・販売が可能。医師発行の処方箋（Prescription）が必要な薬は処方箋なしでは購入不可。
- **ドラッグストア（一般小売薬局）**：OTC（市販薬）のみ販売。Mannings、Watsonsなど。

日本から持ち込んだ処方薬の継続処方を受ける場合、香港の医師に同等薬を再処方してもらうか、日本語対応クリニックで持参薬の確認を受ける方法が一般的。一部の薬（向精神薬等）は香港への持ち込みに事前申告が必要な場合があるため、渡航前に在香港日本国総領事館への確認を推奨する。

## 救急・緊急時の対応

**緊急通報番号は999**（日本の119・110相当の総合番号）。救急（Ambulance）要請もこの番号で受け付ける。英語・広東語での対応が可能。

公立病院の救急外来（A&E）はトリアージ制で、重症患者を優先的に処置する。軽傷での受診は数時間以上の待機が伴うケースが多く、私立病院のA&Eが選択肢となる（費用HKD 2,000〜5,000）。

## 健康診断・ワクチン接種体制

香港では健康診断（Health Check）は私立医療機関で各種パッケージが提供されており、費用はHKD 2,000〜15,000程度（含む内容による）。公立病院での無料健診は基本的に存在しないが、特定のスクリーニングプログラム（大腸がん検診等）は対象者向けに提供されている。

**ワクチン接種：**
- 子どもの定期接種（EPI：Expanded Programme on Immunization）は母子健康院（MCHC）で無料提供
- 成人・渡航前ワクチン接種は私立クリニックで実施（費用HKD 200〜500/回）
- COVID-19、インフルエンザ等は毎年更新されるため、衛生署（Department of Health）の最新情報を参照
