# 홍콩 주거 가이드

홍콩에서 집을 구하는 것은 속도가 빠르고 경쟁이 치열하며 정보가 불투명한 프로세스입니다. 좋은 물건은 1~2일 내에 계약되는 일도 드물지 않고, 계약 의사결정을 극히 짧은 시간 내에 해야 합니다. 임대 흐름, 계약 조항, 주의사항을 이해하면 대부분의 실수를 예방할 수 있습니다.

---

## 1. 주요 집 구하기 채널

| 채널 | 장점 | 단점 |
|---|---|---|
| 유자격 부동산 중개업자(中原/美聯/利嘉閣 등) | 물건 많음; 수속 지원 | 중개 수수료 필요; 성약 우선 업자도 있음 |
| Spacious.hk / 28Hse.com | 집주인 직거래·중개 물건 모두 | 정보가 오래된 것도 있음; 직접 필터링 필요 |
| WeChat/카카오톡 그룹 (한인·재홍콩 외국인) | 주거 지역 정보 집중 | 진위 혼재; 직접 확인 필요 |
| 회사 수배 (입사 시) | 편리; 협상 지원 있음 | 선택지 제한 |
| Facebook (외국인 커뮤니티) | 외국인 집중 지역 정보 | 주로 영어로 운영 |

---

## 2. 집 구하기 흐름

**Step 1: 예산과 지역 확정**
- 월 소득의 30~40%를 임대료 상한으로
- 직장과 자녀 학교에서 지역 좁히기

**Step 2: 내부 확인**
- 중개업자에게 조건 전달 (평형/예산/입주 희망일)
- 보통 2~3일 내에 내부 확인 조율
- 내부 확인 전 해당 물건의 시세 및 건물 정보 조사

**Step 3: 신청과 협상**
- 집주인 제시 가격에서 3~8% 할인 협상이 일반적
- 여러 물건에 동시 신청도 보통
- 의향 확인 후 통상 HKD 5,000~20,000의 성의금 지불

**Step 4: 정식 계약 체결**
- 성의금 지불 후 1~2주 내에 정식 계약
- 계약 시 보증금(통상 2개월분)과 첫 달 임대료 지불

---

## 3. 표준 임대 계약 주요 조항

| 조항 | 내용 |
|---|---|
| 계약 기간 | 통상 2년 |
| 제12개월 해약 조항 | 쌍방이 제12개월에 30~60일 전 서면 통보로 해약 가능 (**매우 중요**) |
| 외교 조항 (Diplomatic Clause) | 고용주 사정에 의한 전근·해고 시 위약금 없이 해약 가능 (사전 협상 필요) |
| 보증금 | 통상 2개월분; 퇴거 시 손상분 제외 후 반환 |
| 수선 책임 | 통상 HKD 500 이하 소수선은 임차인; 대규모 수선은 집주인 |
| 개조 제한 | 벽이나 고정 설비 변경은 통상 금지 |
| 전대 금지 | 통상 무단 전대 불가 |

> **외교 조항에 대해**: 취업 비자로 재홍콩하는 분은 임대 계약에 반드시 Diplomatic Clause 삽입을 협상해야 합니다. 비자나 취업이 갱신되지 않을 경우 위약금 없이 해약할 수 있는 권리입니다. 중개업자가 자발적으로 제안하는 경우는 거의 없으므로 임차인 측에서 요구해야 합니다.

## 4. 보증금과 중개 수수료

| 항목 | 금액 | 비고 |
|---|---|---|
| 보증금 | 통상 2개월분 | 약 HKD 2.5만~6만; 퇴거 시 반환 |
| 중개 수수료 | 통상 1개월분 | 임차인·집주인 절반 또는 전액 집주인 부담도 |
| 첫 달 임대료 | 1개월분 | 계약 시 지불 |
| 인지세 | HKD 5~1,000 (임대료·기간에 따라) | 세무국에 납부; 중개 대행 가능 |
| **초기 비용 합계** | **약 3개월분** | 2BR 물건 기준 HKD 4.5만~9만 정도 |

## 5. 홍콩 주거 유형

| 유형 | 특징 | 적합 대상 |
|---|---|---|
| **사인 옥원(현대 맨션)** | 1970년대 이후 준공; 관리인/시설 있음 | 외국인·재홍콩 한국인의 첫 번째 선택 |
| **唐樓 (당루)** | 오래된 건물(1950~70년대); 엘리베이터 없음; 관리 없음 | 저렴한 임대 희망; 오래된 환경 수용 가능한 분 |
| **촌락 주거 (신계)** | 단독주택 또는 다세대 촌락; 정원·차고 있음 | 넓이와 자연 환경 중시; 차 있는 가족 |
| **서비스 아파트** | 호텔식 관리; 풀 퍼니시드; 광열비 포함; 월 단위 | 착홍 후 1~3개월 과도기에 최적 |

---

## 6. 내부 확인 시 체크포인트

| 확인 항목 | 이유 |
|---|---|
| 실용 면적 (건축 면적의 70~80%) | 홍콩은 건축 면적 표시가 일반적; 실제는 20~30% 작음 |
| 창문 방향과 소음 | 도시부 소음이 크다; 여러 시간대에 확인 |
| 광대역 환경 | 오래된 건물은 광섬유 대응이 제한될 수 있음 |
| 에어컨 완비 여부 | 홍콩 여름은 길고 더움; 후설치는 비용 큼 |
| 수납 공간 | 홍콩 방은 대체로 수납이 적음 |
| 휴대폰 전파 | 지하·주차장 부속 건물에서 전파가 약한 경우 있음 |

---

## 요약

홍콩 임대 시장은 빠르고 경쟁이 치열하지만, 표준적인 절차와 계약 조항을 이해하면 대부분의 리스크를 회피할 수 있습니다. Diplomatic Clause(외교 조항), 실용 면적 확인, 보증금 반환 조건 확인은 처음 임대 시 놓치기 쉬운 3대 중요 포인트입니다.
