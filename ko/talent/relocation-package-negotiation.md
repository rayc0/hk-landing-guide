# 홍콩 부임 패키지 협상 가이드

홍콩 부임 패키지는 기본급뿐만 아니라 '수당 구성'이 총 비용과 삶의 질에 크게 영향을 미칩니다. 특히 주거 수당과 교육 보조는 연간 HKD 100만〜200만（약 1억7,000만〜3억4,000만원）에 달하기도 하며, 협상 여지가 큰 항목입니다.

---

## 1. 표준 부임 패키지 구성

| 항목 | 일반적인 내용 | 협상 우선도 |
|---|---|---|
| 주거 수당 | HKD 20,000〜80,000/월（직위에 따름）| ★★★ 최우선 |
| 자녀 교육 보조 | HKD 80,000〜200,000/년/인（1〜2명까지）| ★★★ 최우선 |
| 귀성 항공권 | 연 1〜2회（이코노미〜비즈니스）| ★★ 중요 |
| 이사 비용 | 한국〜홍콩간 실비（HKD 20,000〜80,000）| ★★ 중요 |
| 착임 정착 수당（Sign-on bonus）| 연수입의 10〜30% 상당（협상 여부）| ★★ 중요 |
| 의료보험 | 본인+가족 그룹 의료보험 | ★ 확인 필수 |
| 귀국 지원（Repatriation）| 임기 종료 시 귀국 이사 비용 | ★ 확인 필수 |

---

## 2. 주거 수당 시세와 협상 포인트

| 직위 레벨 | 주거 수당 시세（월）| 대응 주거 타입 |
|---|---|---|
| 어소시에이트·애널리스트 | HKD 15,000〜25,000 | 신계·구룡 1BR |
| 매니저·부장급 | HKD 30,000〜50,000 | 구룡·홍콩섬 2BR |
| 이사·전무급 | HKD 50,000〜80,000+ | 홍콩섬·타이쿠 2〜3BR |

**세무 주의**:
- 월 현금 주거 수당은 급여로 전액 과세
- 고용주가 직접 주거를 제공하는 경우 급여의 10% 상당만 과세（절세 효과 큼）
- **협상 전략**: '주거 수당을 현금으로'보다 '회사가 직접 물건을 빌려 제공' 형태가 세무상 유리

---

## 3. 교육 보조 협상

자녀가 있는 경우 교육 보조는 최중요 협상 항목:

| 보조 형태 | 내용 | 주의 사항 |
|---|---|---|
| 상한 방식 | 실비 청구·상한 HKD 80,000〜200,000/년 | 영수증·학비 청구서 필요 |
| 고정 수당 방식 | 월 HKD 6,000〜16,000을 급여에 가산 | 과세 대상 |
| 회사 직접 지불 | 학교에 직접 지불（비과세 취급이 되는 경우 있음）| IRD 가이드라인 확인 |

**협상 팁**:
- 자녀의 수와 현재 학년을 사전에 전달
- ESF나 인터내셔널 스쿨의 실제 학비（HKD 130,000〜166,000/년）를 제시
- 'HKD 200,000/년/2명까지'가 대형 다국적 기업의 표준
- 한국인 학교（KISHN）이용 시에도 실비 기준으로 협상 가능

---

## 4. 기타 중요 협상 항목

### 귀성 항공권
- 최소 연 1회 왕복 항공권（본인+가족）확인
- 비즈니스 클래스 적용 직위 기준 명확화
- '플렉시블 티켓'（일정 변경 가능）지정 권장

### 이사 비용
- 착임 시 이사 비용（한국→홍콩）: 실비 정산이 기본
- 가재 임시 보관비나 보험료 포함 여부 확인
- 임기 종료 귀국 이사 비용도 서면에 명기

### 착임 수당（Sign-on Bonus）
- 이직 리스크 보상으로 초봉과 함께 협상
- 보통 1년 이내 퇴직 시 반환 의무 있음（서면 확인）

---

## 5. 협상 타이밍과 주의 사항

| 타이밍 | 권장 행동 |
|---|---|
| 오퍼 레터 수령 전 | '기대하는 수당 구성'을 미리 전달 |
| 오퍼 레터 수령 후 | 1주일 이내에 서면으로 조건 협상 |
| 서명 전 | 전 조건 서면 확인 + 사내 변호사 체크 |

**한국인이 주의할 협상 방식**:
- '한국 회사에서는 이랬다'는 비교는 역효과
- 감정적 협상보다 '시장 시세'와 '생활비 실태'를 제시하는 것이 효과적
- 세세한 항목을 한꺼번에 많이 요구하면 인상 나쁨（우선 순위 2〜3항목에 집중）
- 대기업 한국인 선배 주재원에게 미리 실상을 들어두는 것이 강력한 사전 준비

- [ ] 귀성 항공권 횟수·클래스·대상자（가족 포함 여부）
- [ ] 착임 이사 비용·귀국 이사 비용 부담 범위
- [ ] 의료보험 보상 내용·가족 적용
- [ ] Sign-on bonus 반환 조건과 기간
- [ ] 임기·연장·조기 귀국 조건

## 요약

홍콩 부임 패키지에서 주거 수당과 교육 보조가 금액 임팩트가 가장 큰 협상 항목입니다. 특히 주거 수당의 '직접 제공 vs 현금 수당' 지불 형태 차이는 세무적으로 연간 수백만 원의 차이가 됩니다. 협상은 감정이 아닌 시장 데이터로 진행하고, 우선 항목을 좁혀 서면으로 확정시키는 것이 중요합니다.
