# 香港稅務指南：新移民的薪俸稅、利得稅及強積金全解

香港是全球個人及企業稅務負擔最低的司法管轄區之一。新移民往往驚喜地發現：**無資本增值稅、無消費稅（GST/VAT）、無股息預扣稅、無遺產稅**，加上全球最低的個人入息稅率之一。了解香港稅制的運作方式，讓你從抵港第一天起就能作出明智的財務決策。

---

## 香港稅制的核心：地域來源原則

令香港稅制如此吸引的根本原則：**香港只對在香港產生或得自香港的收入徵稅**，即地域來源原則。

實際意義：
- 在香港履行職務所賺取的收入須課稅
- 海外工作收入、海外投資收益、境外業務收入**不須**繳納香港稅
- 深圳公司支付的薪酬一般不屬香港薪俸稅範疇
- 海外股票股息及海外物業租金收入——不課稅

這使香港對擁有跨境收入的國際專才、服務全球客戶的自由職業者，以及在多個司法管轄區運營的企業主尤其有利。

## 薪俸稅：如何計算

### 誰須繳納薪俸稅

任何在香港受僱工作、擔任職位或領取退休金的人，均須就其香港來源收入繳納薪俸稅。「在香港產生」按服務履行地點而非僱主所在地決定。

### 稅率

香港薪俸稅採用**累進稅率**，最高17%，但因有慷慨的免稅額，實際稅率通常遠低於此。

**累進稅率（適用於扣除免稅額後的應課稅入息）：**

| 應課稅入息 | 稅率 |
|---|---|
| 首HK$50,000 | 2% |
| 其後HK$50,000 | 6% |
| 其後HK$50,000 | 10% |
| 其後HK$50,000 | 14% |
| 其餘部分 | 17% |

**標準稅率上限**：稅款不超過**入息淨額（扣除免稅額前）的15%**（高收入者取兩者較低者）。

以實例說明：年收入HK$60萬（月薪約HK$5萬）人士，享有標準免稅額後，實際稅率通常約8-10%——遠低於英國、澳洲或加拿大同等收入的稅負。

### 個人免稅額

以下免稅額可減低應課稅入息：

| 免稅額 | 金額（2025/26） |
|---|---|
| 基本個人免稅額 | HK$132,000 |
| 已婚人士免稅額 | HK$264,000 |
| 子女免稅額（每名） | HK$130,000 |
| 供養父母免稅額（每名） | HK$25,000-50,000 |
| 長者住宿照顧開支 | 最高HK$100,000 |
| 進修開支 | 最高HK$100,000 |
| 居所按揭利息 | 最高HK$100,000 |
| 強積金自願供款 | 最高HK$18,000 |

基本個人免稅額即表示：年收入低於HK$132,000的人士**毋須繳納**薪俸稅。

### 稅務年度及報稅

香港稅務年度為**4月1日至翌年3月31日**。稅務局每年5月向納稅人發出個人報稅表，申報上一年度收入。**首年新抵港人士**常收到「暫繳稅」評稅通知——這是預繳下年度稅款，非重複徵稅。

## 利得稅：適用於自僱及業主

若你在香港經營業務（包括自由職業、顧問服務或獨資經營），業務收入須繳納**利得稅**而非薪俸稅。

### 稅率

| 業務類型 | 利得稅率 |
|---|---|
| 非法團業務（首HK$200萬利潤） | 7.5% |
| 非法團業務（逾HK$200萬部分） | 15% |
| 法團（首HK$200萬利潤） | 8.25% |
| 法團（逾HK$200萬部分） | 16.5% |

2018年引入的兩級制稅率令中小企業大為受益——**首HK$200萬企業利潤只須繳納8.25%**，在國際上屬極低水平。

### 境外利潤豁免

**完全在香港境外進行**的業務活動所產生的利潤，可申請豁免利得稅（境外申索）。這是合法且廣泛使用的稅務安排——但須有充分文件證明相關業務活動（洽商、簽約、決策）確實在香港境外進行。稅務局自2023年起依據新的境外收入豁免（FSIE）規則，對境外申索審查更為嚴格。

## 強積金（MPF）：香港強制退休保障計劃

強積金是香港強制性退休儲蓄計劃，僱主及僱員均須供款。

### 供款比率

| 供款方 | 供款比率 | 上限 |
|---|---|---|
| 僱員 | 相關入息的5% | 每月最高HK$1,500 |
| 僱主 | 相關入息的5% | 每月最高HK$1,500 |

**相關入息上限**：供款按月薪HK$30,000計算上限，超過部分不須額外供款，僱員及僱主各固定最多HK$1,500。

**最低入息門檻**：月薪低於HK$7,100的僱員可免除僱員供款（僱主仍須供款）。

### 新移民的強積金事項

- 受僱後須在60天內加入強積金計劃
- **豁免情況**：外籍家庭傭工、合約少於13個月的境外僱員（可申請豁免）、有獨立退休計劃的公務員
- 強積金戶口資金**鎖定至65歲**方可提取（例外情況：末期疾病、永久離港、小額結餘）

### 自願供款（可扣稅）

你可在強制5%之外額外供款（自願供款），每年最多HK$18,000可從薪俸稅應課稅入息中扣除——是立竿見影的合法節稅方式。

## 物業稅

若你在港擁有出租物業，租金收入須繳納**物業稅**，稅率為應評稅淨值的15%（淨值為租金收入扣除20%法定修葺及支出免稅額後的金額）。同時為薪俸稅納稅人的業主，可選擇以個人入息課稅方式評稅，若本身有足夠免稅額，整體稅款可能因此減少。

---

## 無財富稅、資產增值稅及遺產稅

確認大眾印象屬實：香港**確實沒有**：
- 資產增值稅（出售股票、物業或其他資產的利潤不課稅）
- 股息預扣稅（香港公司向非居民派發的股息不課稅）
- 遺產稅（2006年廢除）
- 商品及服務稅（GST）或增值稅（VAT）

---

## 避免雙重課稅協議

香港已與超過45個司法管轄區簽訂全面避免雙重課稅協議（CDTA），包括中國內地、英國、日本、法國及荷蘭等。若你的收入在另一地已被課稅，同時在技術上亦屬香港來源收入，這些協議可防止雙重課稅。

---

## 新移民實用貼士

**從第一天起妥善保存薪酬單及稅務紀錄。** 稅務局或會要求追溯數年的文件，尤其當你有香港及海外混合收入時。

**了解僱傭合約的薪酬結構。** 外派人員待遇通常包括住房津貼、子女學費等福利——這些通常亦屬薪俸稅評稅範疇。

**第一年可考慮聘請稅務顧問。** 香港稅制整體簡單，但第一年的暫繳稅計算及部分年度居港規則可能令人困惑。第一年聘請具資格的香港執業會計師，費用通常物有所值。

## 總結

| 稅種 | 稅率 | 重要說明 |
|---|---|---|
| 薪俸稅 | 累進2%-17% / 標準稅率15% | 地域來源；慷慨免稅額降低實際稅率 |
| 利得稅（法團） | 8.25%-16.5% | 兩級制；境外收入可能豁免 |
| 利得稅（非法團） | 7.5%-15% | 兩級制 |
| 物業稅 | 應評稅淨值15% | 僅適用於租金收入 |
| 資產增值稅 | 0% | 香港無資產增值稅 |
| 遺產稅 | 0% | 2006年廢除 |
| 強積金 | 僱主5% + 僱員5% | 每方每月上限HK$1,500 |

*本文提供一般資訊，稅務規則可能變更，個別情況各有不同。請向具資格的香港稅務顧問或稅務局查詢適用於你具體情況的建議。*
